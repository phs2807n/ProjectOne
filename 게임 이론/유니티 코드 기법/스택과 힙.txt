Stack(스택) 메모리
 - 함수가 실행될 때 기본적으로 확보되는 메모리
 - 크기가 상대적으로 작다.
 - 함수 실행코드, 함수 파라미터, 지역변수
 - 사용하는데 속도가 빠르다.


Heap(힙) 메모리
 - 컴퓨터에 남아있는 렘
 - 크기는 보통 무한으로 처리
 - 동적(프로그램 실행 중) 할당 용도로 사용
 - 속도가 느리다.


C# 데이터 타입

 - 값 타입
   ㄴ 스택메모리에 저장
   ㄴ unmanaged type
   ㄴ 깊은 복사(완전한 사본을 가지게 된다.)
   ㄴ struct
 

 - 참조 타입(Reference Type)
   ㄴ 힙 메모리에 저장
   ㄴ managed type(가비지 콜랙팅이 된다.)
   ㄴ 얕은 복사(복사 속도가 빠르다.)
   ㄴ class
